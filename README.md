# Evaluacion-3-python-flask
Proyecto de evaluación 3, introducción a python - aplicación web con flask

## Descripción

Esta aplicación web implementa dos ejercicios:

 Cálculo de notas: Permite calcular el promedio de notas de un estudiante.
Búsqueda del nombre más largo: Encuentra el nombre más largo de una lista de nombres.

  ## Estructura del proyecto

*   `evaluacion3.py`: Archivo principal con la lógica de la aplicación Flask.
*   `templates/`: Carpeta con los archivos HTML.
*   `static/`: Carpeta con los archivos CSS (si los hay).
*   `info_proyecto.txt`: Archivo con información adicional del proyecto.
